  12. TLB or Translation Lookaside Buffer ch.3.1.1 is a memory management unit for improving the virtual address to physical address translation cache speed. TLB is a small, a virtual address cache, where each row holds a block composed of a single PTE. If there is no TLB, then twice each taking data needs to access memory, that check page table to get the physical address and fetch data.

    The first CPU appled TLB in the CPU X86 system was Intel-486 CPU. In the CPU inside the X86 system, generally have the following four groups TLB:

    • Group I: General page table cache (4K byte page) instruction page table cache (Instruction-TLB);
    • Group II: General cache page table (4K-byte pages) data page table cache (Data-TLB);
    • Group III: Large cache page table (2M/4M-byte page) instruction page table cache (Instruction-TLB);
    • Group IV: Large cache page table (2M/4M-byte page) data page table cache (Data-TLB);
